void Scene::loadScene ( std::string  sceneFile)

Loads a scene using a specified scene file (.jes)

Scene file format is as follows: List of meshes List of shaders List of textures List of materials Shader Shader variables List of MeshRenderers Mesh Material Position

Specifically, m = mesh, s = shader, t = texture, a = material r = meshrenderer

m filename.extention s vertName fragName t texName.extention a shaderName otherVars(?) void *

/param sceneFile The location, relative to the working directory, of the scene we want to load.

void Scene::saveScene ( std::string  filename)

Saves the scene cache and other scene data to the specified file.

This function is intended for use when you are creating a new scene file since it saves the name of every object in the cache, plus the geometric data and material of any meshes plus extra data to be defined in the future. To save a scene for use in a savegame, use Scene.saveSceneChanges()

Also note that this function uses the overloaded stream insertion operator defined by each class.

Parameters
fileNameThe filename where you want to save the scene. This is assumed to be relative to the working directory
